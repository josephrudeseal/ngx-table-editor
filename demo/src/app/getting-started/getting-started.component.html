ngx-table-editor

A library that transforms dorky static HTML tables into cool dynamic editable components. Tested with Angular 6.1.

Getting started


1. Install the library.

```console npm install --save-dev '@ngx-table-editor/core' ```

2. Import the library in your module.


3. Transform some dull HTML tables.

See the examples or read the API docs


4. Optional: Consider to install the extras module and import from it whatever you want to use.
