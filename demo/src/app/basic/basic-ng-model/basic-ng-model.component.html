Minimal Example: Using NgModel

The most minimal set-up, in the template-driven way

An example using only the basic directives tableEditor, teRow and teCell. Click a row in the table to start editing, then use the tab, shift-tab, arrow up and arrow down keys to navigate through the table. Hit enter or esc to exit the editor. Don't forget to import FormsModule.
